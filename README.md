# Facial-Emotion-Detector-using-ResNet

## ✏️ Overview

Facial Emotion Detection is a computer vision application that identifies human emotions (such as happiness, sadness, anger, surprise, fear, disgust, and neutrality) from facial images. This project leverages ResNet(Residual Networks), a deep convolutional neural network architecture, to achieve high accuracy in emotion classification.

## 📜 Requirements

##### 1. Python
##### 2. PyTorch
##### 3. OpenCV
##### 4. NumPy, Pandas, Matplotlib
##### 5. TensorFlow
##### 6. MTCNN

## 📋 Methodology

- Architecture: ResNet-50 were fine-tuned for emotion classification
- Loss Function: Categorical Cross-Entropy
- Optimizer: Adam
- Data Augmentation: Random flips, rotations, and normalization to improve generalization

## 🌟 Future Improvements
